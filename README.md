# 🏥 Sistema CRUD de Pacientes

API RESTful desarrollada en PHP para la gestión de pacientes, implementando operaciones CRUD completas, validación de datos y arquitectura básica MVC.

---

## 🚀 Tecnologías utilizadas

- PHP 8 (PDO)
- MySQL
- Docker
- JavaScript (Fetch API)
- Bootstrap 5

---

## ⚙️ Funcionalidades

- ✅ Crear pacientes
- ✅ Listar pacientes
- ✅ Actualizar pacientes
- ✅ Eliminar pacientes
- ✅ Validación de datos
- ✅ Protección contra SQL Injection (PDO)
- ✅ Arquitectura MVC básica

---

## 🐳 Ejecución del proyecto

```bash
docker compose up -d --build
```

## Endpoints

GET /backend/index.php

POST /backend/index.php

PUT /backend/index.php?id=1

DELETE /backend/index.php?id=1
